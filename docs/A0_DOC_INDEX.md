本文是 `ah-disclosure-kit` 的中文文档入口，便于快速定位各类说明文档。

## 1. A 类：使用入口

- `A1_INSTALLATION_AND_USAGE.md`：安装、MCP 注册、常用命令。
- `A2_UPDATE_LOCAL_INSTALL.md`：本地更新、Skill 同步、MCP 重启。
- `A3_WORKFLOW.md`：用户提问后的完整处理链路。
- `A4_MCP_TOOLS.md`：MCP 函数清单和用途说明。

## 2. B 类：数据和 PDF

- `B1_PDF_INGEST.md`：PDF 下载后如何解析、检索、OCR、导出。
- `B2_COMPANY_DATA.md`：A/H 股结构化公司数据来源。
- `B3_HKEX.md`：港股 HKEXnews、stockId 和披露文件查询说明。
- `B4_PROSPECTUS.md`：招股书、上市文件、募集说明书路径。

## 3. C 类：测试

- `C1_TEST_PLAN.md`：本地测试和真实数据测试建议。

## 4. D 类：开发定稿

- `D1_DEVELOPMENT_PLAN_V1_0.md`：v1.0 设计、边界、目录、数据流和后续规划。

## 5. 其他文件

- `README.md`：项目总览，供包管理器和 GitHub 默认展示。
- `CHANGELOG.md`：版本更新记录。
- `skills/ah-disclosure/SKILL.md`：Claude Code / Codex 使用本工具时读取的 Skill 规则。
- `examples/A0_CLAUDE_CODE_COMMANDS.md`：Claude Code 常用命令和提示词示例。
- `scripts/INSTALL_AND_CHECK.ps1`：Windows 一键安装/检查脚本，只安装 Python 包依赖，不安装 Python 或 Tesseract。
